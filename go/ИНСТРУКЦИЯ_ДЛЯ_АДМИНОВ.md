# Бот-антиспам: коротко

Сам удаляет и банит: флуд, известный спам, опасные ссылки, сообщения с плохим контекстом (скам/казино/интим со ссылкой, `@username` или призывом писать в ЛС), запрещённые слова (если включили фильтр). Может работать жёстко (бан) или мягко (только удаление), в зависимости от настроек. Владельца чата не банит (запретить не может), но его сообщения тоже проверяет.

## Команды

- `/help` — список функций
- `/report` (ответом на сообщение) — пожаловаться, после нескольких жалоб автора забанят
- `/ban` (ответом на сообщение) — забанить сразу
- `/addspam <слово>` / `/removespam <слово>` / `/triggers` — своё слово в фильтр / убрать / посмотреть список
- `/addcorewords mat|insults|spam|scam` — добавить готовый набор слов
- `/contextmode ban|delete|off|default` — как встроенный фильтр плохого контекста работает в этом чате
- `/allowcontext <фраза>` / `/unallowcontext <фраза>` / `/allowcontexts` — исключения для встроенного фильтра этого чата
- `/blockdomain <домен>` / `/unblockdomain <домен>` / `/domains` — блок ссылок на домен

## Подключение к новой группе

1. Добавить бота в группу.
2. Сделать его админом.
3. Включить права **Delete messages** и **Ban users** — без них не работает.
4. Если нужна капча для новых участников — ещё и право **Restrict members**.

## Быстрые сценарии

- Много серого рекламного мусора, но не хочется сразу банить: `/contextmode delete`
- Нужно жёстко вычищать скам и казино: `/contextmode ban`
- Фильтр удаляет полезную фразу вашего чата: `/allowcontext <эта фраза>`
- Хотите вернуться к глобальному поведению из `.env`: `/contextmode default`
